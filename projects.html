AutoVision: Car Model Code Identifier

Python, TensorFlow, Keras, MobileNetV2, OpenCV, AWS

Designed a visual recognition system that identifies a car's model code from a single image, supporting applications like automated listing verification and vehicle part compatibility. This project required curating and cleaning real-world image data, conducting extensive research to identify the most suitable model architecture and training strategies, and evaluating accuracy under varied lighting and viewing conditions.

ASLTranslator: American Sign Language Recognition App

Python, TensorFlow, OpenCV, MediaPipe, Tkinter

Developed an application that translates ASL finger spelling into text using real-time webcam input, supporting accessible communication and interactive learning tools. The project involved collecting and processing hand landmark data, researching sequence modeling techniques, and optimizing for performance and usability in a live user environment.
